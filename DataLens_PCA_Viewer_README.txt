Description:
------------
The DataLens Free PCA Viewer is a lightweight desktop tool designed to visualize hyperspectral data from ENVI .hdr files.
It helps researchers explore spectral variance and identify key regions using Principal Component Analysis (PCA).

How to Use:
-----------
1. Download and extract the ZIP file.
2. Double-click on 'DataLens_PCA_Tool.exe' to start the app.
3. Click 'Browse' to upload your ENVI .hdr file.
4. Use the available buttons to visualize PCA components and spectra.
